Dynamic Geometries: The Chladni-Möbius Visualizer

Executive Summary

The Chladni-Möbius Visualizer is an interactive WebGL application that combines principles from physics and mathematics to render complex geometric transformations. By merging Chladni patterns and Möbius transformations—enhanced with noise modulation—this tool produces visualizations that respond to user input and evolve over time.

Introduction

This visualizer demonstrates how sound-induced wave patterns (Chladni patterns) and complex mathematical mappings (Möbius transformations) can be integrated into a single interactive system using WebGL via Three.js.

Features:

  • Educational Exploration: Visualizing and understanding mathematical transformations.
  • Digital Art Creation: Generating unique, parameter-driven visuals.
  • Technical Demonstration: Showcasing advanced WebGL capabilities.
  • Practical Application: Applying theoretical math in visual computing.

The Core Concepts

🌀 Chladni Patterns

Chladni patterns are visual manifestations of standing waves—seen when vibrating plates gather sand at nodes where wave interference cancels out. In this project, we simulate them in real-time via shaders to explore dynamic symmetry and wave interference patterns.

♾️ Möbius Transformations

Defined as:

f(z) = (az + b) / (cz + d)  where ad − bc ≠ 0

These are conformal maps on the complex plane, preserving angles and local geometry.

Applications:

  • Texture warping
  • Optical lens simulation
  • Shader-based deformations
  • Fractal generation

Combining Physics and Mathematics

We integrate the two systems by:

  • Using wave dynamics to create surface variations
  • Twisting the geometry with Möbius transformations
  • Modulating with noise for natural variation

🎛️ Interactive Features

Users can control:

  • Chladni: Amplitude, frequency, and time scale
  • Möbius: Coefficients a, b, c, d, factor, compensation
  • Noise: Scale and time
  • Rotation: Enable/disable, frequency, thresholds

Presets include:

  • Default
  • No Rotation
  • Strong Waves
  • Classic Möbius
  • Organic Motion

vec2 complex_mul(vec2 a, vec2 b) {
  return vec2(a.x * b.x - a.y * b.y, a.x * b.y + a.y * b.x);
}

vec2 complex_div(vec2 a, vec2 b) {
  float denom = b.x*b.x + b.y*b.y;
  return vec2((a.x*b.x + a.y*b.y)/denom, (a.y*b.x - a.x*b.y)/denom);
}
